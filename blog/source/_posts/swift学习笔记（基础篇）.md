####一数值型字面量
* 一个十进制数，没有前缀
* 一个二进制数，前缀是0b
* 一个八进制数，前缀是0o
* 一个十六进制数，前缀是0x

列如：下面的所有整数字面量的十进制值都是17
>let decimalInteger = 17
let binaryInteger = 0b10001       // 二进制的17
let octalInteger = 0o21           // 八进制的17
let hexadecimalInteger = 0x11     // 十六进制的17
